# RPGUser

层级结构：用户、英雄、装备、宝石、宠物
用户：
    基础属性：等级，生命值
    高级属性：战斗力
英雄：
    基础属性：等级，生命值
    高级属性：战斗力，最大生命值
装备：
    基础属性：基础攻击
    高级属性：实际攻击力
宝石：
    基础属性：攻击力
宠物：
    基础属性：战斗力

战斗力公式：用户战斗力 = 在队列中的英雄的战斗力总和 + 用户等级 * 3
           英雄战斗力 = 最大生命值 * 1.5 + 英雄攻击力 * 1.8
           英雄最大生命值 = 英雄等级 * 100 * 英雄品质
           英雄攻击力 = 英雄装备攻击力总和
           装备攻击力 = 基础攻击力 + 宝石攻击力
